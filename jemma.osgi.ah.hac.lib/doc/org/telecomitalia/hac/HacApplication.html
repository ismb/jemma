getCategories

java.util.Vector getCategories()
Returns the categories supported by this driver. A category permits to specify the purpose of the device. For instance a Smart Meter may returns the list of appliances it can be connected to: TV, Oven, Lamp and so forth. A temperature sensor may specify a category list or not. There is an Unknown category.

Returns:
A Vector of Categories or null, if the Driver doesn't specify any particular category. A null category list should be handled accordingly by the caller.
